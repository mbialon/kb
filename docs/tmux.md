# tmux

| Action                       | Shortcut                |
| ---------------------------- | ----------------------- |
| Create new window            | C-b c                   |
| Kill current window          | C-b &                   |
| Rename window                | C-b ,                   |
| Split vertically             | C-b %                   |
| Split horizontally           | C-b : split-window      |
| Resize pane by 20 lines down | C-b : resize-pane -D 20 |
